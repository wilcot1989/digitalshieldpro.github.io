## 2. Two-Factor Authentication (2FA)

Enable 2FA on every account, especially:
- Email (Gmail, Outlook)
- Cloud storage (Google Drive, Dropbox, OneDrive)
- Banking and financial services
- Social media
- Freelance platforms (Upwork, Fiverr, etc.)
- Code repositories (GitHub, GitLab)

**Use an authenticator app** (Google Authenticator, Authy, or your password manager's built-in 2FA) â€” NOT SMS-based 2FA, which can be intercepted via SIM swapping.

## 6. Client Communication Security

### For Sensitive Conversations
- Use **Signal** for encrypted messaging
- Use **ProtonMail** for encrypted email
- Never share passwords or credentials via regular email
- Use your password manager's secure sharing feature instead

### File Sharing
- Use **encrypted cloud storage** (Tresorit, ProtonDrive) for sensitive client files
- Set expiration dates and access controls on shared links
- Never use personal email for professional file sharing

## 7. Device Security Basics

### Laptop Security
- Enable **full disk encryption** (BitLocker on Windows, FileVault on Mac)
- Set a strong login password (not just a PIN)
- Enable **automatic screen lock** (2 minutes or less)
- Keep OS and all software updated
- Enable **remote wipe** capability (Find My Mac, Find My Device)

### Phone Security
- Use biometric + strong PIN (6+ digits)
- Enable remote wipe
- Keep apps and OS updated
- Do not install apps from unknown sources
- Use your VPN when on public Wi-Fi
